## 언어 및 커뮤니케이션 규칙
- 기본 응답 언어: 한국어
- 코드 주석: 한국어로 작성
- 커밋 메시지: 한국어로 작성
- 문서화: 한국어로 작성
- 변수명/함수명: 영어 (코드 표준 준수)

## 마크업 코딩 스타일
- 들여쓰기: 2칸
- BEM 구조

## 추가 선호사항
- Tailwind CSS 사용
- TypeScript 사용
- 컴포넌트 분리 및 재사용
- 반응형 필수

## 개발 환경
- OS: macOS

## 프로젝트 마무리 루틴
사용자가 `/last` 명령어를 입력하면 "프로젝트 마무리 루틴"을 실행한다:
1. `code-reviewer` 서브에이전트로 전체 코드 리뷰
2. Playwright MCP로 주요 UI 검증
3. 종합 최종 리포트 작성
자세한 실행 지침은 `~/.claude/commands/last.md` 참고.
